# Ontological Physics

Interactive Jupyter Book exploring **Ontological Physics** — fundamental physics rebuilt from ontological principles.

## Core Idea

Everything moves at the speed of light *c* through spacetime.
Only the direction — between space and time — differs.

$$v_{space}^2 + v_{time}^2 = c^2$$
